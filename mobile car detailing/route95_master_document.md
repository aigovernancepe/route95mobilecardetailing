## 2. COMBINED METHODOLOGY

This implementation uses three complementary optimization methods:

### A. Caleb Method (Local SEO Foundation)

**Purpose:** Rank in Google Maps Pack (top 3 local results)

| Element | Implementation |
|---------|----------------|
| GBP Categories | Primary + 2 secondary categories |
| GBP Services | 29 services with descriptions |
| Core 30 Website | Homepage + Category pages + Service pages |
| Title Tags | [Service] + [City] format |
| Entity Relevance | Website structure mirrors GBP exactly |
| Local Links | Chambers of commerce, sponsorships |

### B. Capsule Method (AI Citations)

**Purpose:** Get cited by ChatGPT, Perplexity, Google AI Overviews

| Element | Implementation |
|---------|----------------|
| Question H2s | Frame headings as questions users ask |
| Answer Capsules | 120-150 character direct answers under H2s |
| Link-Free Passages | No hyperlinks in quotable text |
| Original Data | Include unique insights/statistics |
| Clear Structure | Lists and tables for easy extraction |

### C. Gemini/Maps Method (Voice Discovery)

**Purpose:** Get recommended by Gemini during navigation

| Element | Implementation |
|---------|----------------|
| Conversational GBP | Description written as natural speech |
| Contextual Services | Service descriptions answer spoken questions |
| Specificity | Include time estimates, who it's for, context |
| Natural Language | Match how people talk, not keyword stuffing |

### Combined Formula

```
Google Maps Ranking (Caleb)
        +
AI Citations (Capsule)
        +
Voice Discovery (Gemini)
        =
Maximum Local Visibility
```
